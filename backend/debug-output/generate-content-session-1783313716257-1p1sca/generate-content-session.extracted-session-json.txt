  "theory": {
    "overview": "Motion is a fundamental concept describing change. However, it is not absolute; an object's state of rest or motion changes depending on the observer's position (reference point). We categorize motion into two main types: Uniform Motion (constant speed) and Non-Uniform Motion (changing speed), which can be visualized using distance-time graphs.",
    "keyPoints": [
      "Motion is relative to a reference point; an object at rest for one observer may be moving for another.",
      "Uniform motion involves covering equal distances in equal time intervals, resulting in constant speed.",
      "Non-uniform motion involves unequal distances in equal time intervals, indicating changing speed (acceleration/deceleration).",
      "Distance-time graphs provide a visual representation: straight lines indicate uniform motion or rest, while curves indicate non-uniform motion."
    ],
    "detailedContent": "To understand the physical world, we must first define what it means to move. In physics, an object is said to be in 'motion' if its position changes with respect to a fixed point (reference point) over time. Conversely, if the position does not change relative to that reference point, the object is at 'rest'. A classic example involves passengers on a moving bus: they are at rest relative to each other and the driver, but in motion relative to the trees outside. This relativity is crucial for understanding all subsequent physics concepts.\n\nOnce we establish this baseline, we look at *how* objects move. If an object covers equal distances in equal intervals of time (e.g., 10 meters every second), it exhibits Uniform Motion. The speed remains constant throughout the journey. On a distance-time graph, this appears as a straight diagonal line.\n\nIn contrast, most real-world motion is Non-Uniform. Here, an object covers unequal distances in equal intervals of time because its speed changes (it speeds up or slows down). For instance, a car approaching a traffic light decelerates; the distance it covers each second decreases. On a graph, this appears as a curved line where the slope changes."
