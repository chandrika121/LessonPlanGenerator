    "detailedContent": "We begin by defining a pure substance as matter with identical particles throughout. Elements cannot be broken down further chemically; compounds can only be separated into elements via chemical reactions. In contrast, mixtures are physical combinations where components retain their individual properties. We will explore separation techniques: Filtration (separating insoluble solids from liquids), Evaporation/Crystallization (recovering solutes like salt), Simple Distillation (for miscible liquids with large boiling point differences >25°C), and Chromatography (separating pigments based on differential adsorption). Finally, we differentiate colloids (e.g., milk) which scatter light (Tyndall effect) from true solutions."
